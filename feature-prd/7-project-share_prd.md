# Fast Hardware - 方案分享与导出 PRD（7-project-share）

## 📌 文档定位

- 本文件仅定义 **画布侧「分享」→ 分享预览 → 长图/方案包导出** 的产品范围与验收标准。
- **在线创客集市**（列表、鉴权、OSS、审核、管理端）见同目录 **`8-maker-marketplace_prd.md`**，本文件 **不** 重复展开。
- **产品关系类比**：**电路设计画布 + 项目管理 + AI 辅助** 为「工具侧」；分享导出为 **向社区或其它渠道供稿** 的前置步骤（社区侧见文档 8）。

### 与既有 PRD 的关系

| 关联文档 | 说明 |
|----------|------|
| **`8-maker-marketplace_prd.md`** | 创客集市、Supabase、OSS、OAuth、审核与「社区管理」页 |
| **`2-circuit_prd.md`** | 画布、项目 JSON、连线与导出物数据源 |
| **`6-project-isolation_prd.md`** | 分享物绑定 **当前活动项目** `projectId` 与快照 |
| **`3-skills_prd.md` / `4-agent-loop_prd.md`** | 方案简介/BOM 等可复用既有分析能力 |

---

## 1. 背景与问题

1. 用户完成电路与固件后，需要 **结构化、可编辑** 的 **分享预览**，再导出 **长图** 与可选 **方案包**，便于投稿、教学或同步至 **创客集市**（文档 8）。
2. 教育/创客场景需要 **Learning by doing**：展示侧依赖 **缩略图 + BOM + 代码片段** 的可编辑拼装（导出前可改）。

---

## 2. 目标与阶段（本文件范围）

### 2.1 本文件内目标（P0）

1. **电路设计画布页** 提供 **分享** 入口 → **分享预览窗口** → 四块内容可编辑 → **导出长图 PNG**（必选）+ 可选 **方案包 ZIP**（供集市上传或本地复刻）。
2. **电路区块** 在预览内支持 **平移/缩放/重置构图**，导出裁切结果作为 **集市卡片缩略图源**（与文档 8 中 OSS 对象字段对齐即可）。

### 2.2 不在本文件范围

- 用户鉴权、列表拉取、OSS 上传、审核流、管理页 —— 全部由 **`8-maker-marketplace_prd.md`** 覆盖。

---

## 3. 功能需求

### 3.1 电路画布页：分享入口（FR-SHARE-1）

| 项 | 说明 |
|----|------|
| **位置** | 画布区域 **右下角**，不遮挡主操作区；小屏可收入「更多」。 |
| **形态** | **Icon 按钮**，Tooltip「分享方案」；**`data-icon="share-2"`** → **`assets/icon-share-2.svg`**（见 **`assets/README.md`**）。 |
| **行为** | 打开 **分享预览**（模态）；未保存项目时 **强提示保存** 或明确「以内存快照为准」。 |
| **与项目隔离** | 绑定当前 `projectId` 与 `circuit_config` / 固件快照（**`6-project-isolation_prd.md`**）。 |

### 3.2 分享预览窗口（FR-SHARE-2）

#### 四块内容（自上而下长图顺序）

| 顺序 | 区块 | 默认数据源 | 可编辑 |
|------|------|------------|--------|
| 1 | 方案简介 | 项目名 + README 摘要等 | 是 |
| 2 | 方案 BOM 表 | 画布解析物料表 | 是（不要求与画布双向联动） |
| 3 | 电路原型图 | 画布快照 + 视窗内平移缩放 | 仅构图，不改连线 |
| 4 | 代码预览 | 主 `.ino` 等 | 是 |

#### 电路视窗与导出

- **平移 / 缩放 / 重置**；导出第三块与 viewport 一致；**缩略图** 与第三块同源或按比例衍生。
- **长图 PNG**：建议固定宽度 1080～1200px；**方案包 ZIP**：`circuit_config.json`、缩略图、`manifest.json`、代码等（字段命名与文档 8 的 `posts`/存储键约定一致更佳）。

---

## 4. 非功能需求（NFR）

| 类别 | 要求 |
|------|------|
| **导出稳定性** | 大 BOM、长代码不崩溃；失败可重试 |
| **主题** | 预览与导出图在浅/深色下可读 |
| **模块化** | 合成逻辑与「上传集市」解耦，便于文档 8 调用同一 manifest |

---

## 5. 验收标准（P0）

1. 画布右下角 **分享** icon 符合 **`assets/README.md`**。
2. 预览四块 **可见且可编辑**（除电路仅改构图）。
3. 电路区 **拖拽平移 + 滚轮缩放 + 重置**，导出与视窗一致。
4. **PNG 长图** 可保存到用户选定路径；成功有反馈。
5. 未保存项目行为符合 §3.1。

---
