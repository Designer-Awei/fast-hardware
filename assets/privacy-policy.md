# Fast Hardware 隐私政策

更新日期：2026-04-23

本政策说明 Fast Hardware 在账号、登录、社区与本地项目场景下对信息的处理方式。

## 1. 我们收集的信息

在当前版本中，可能涉及以下信息：

1. 账号信息：邮箱地址、昵称、登录方式、用户角色。
2. 会话信息：登录态与必要的本地会话缓存。
3. 本地项目信息：为展示“我的方案”列表，应用会读取你配置的项目目录及其中的项目摘要文件。
4. 社区内容信息：你主动上传、发布、点赞、收藏、复刻或参与审核时产生的数据。

## 2. 信息用途

这些信息主要用于：

1. 完成账号注册、登录与身份识别。
2. 在应用重启后恢复登录状态。
3. 展示个人中心、本地项目列表、创客集市互动状态与管理员审核能力。
4. 保障平台安全、风控与审计。

## 3. 信息存储

1. 本地会话缓存会保存在应用本地目录中，用于恢复登录状态。
2. 社区相关元数据可能存储于 Supabase。
3. 图片、长图、方案包等大文件后续可能存储于对象存储服务。

## 4. 信息共享

除法律法规要求、平台安全治理需要，或为实现你主动发起的社区发布与分享功能外，我们不会无故对外公开你的个人信息。

## 5. 你的权利

你可以：

1. 主动退出登录。
2. 停止使用社区功能。
3. 在后续版本中申请修改或删除相关账号资料与社区内容。

## 6. 政策更新

随着账号体系、创客集市与云服务能力的持续完善，本政策可能更新。更新后将以应用内或仓库中的最新版本为准。
